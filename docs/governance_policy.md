# Governance Policy

This project follows the spirit of SR 11-7 model-risk management: clear purpose, sound development, independent validation, ongoing monitoring, documentation, effective challenge, inventory control, and change management.

No model is production-ready until the model card, validation report, monitoring plan, prohibited-use language, human-review policy, and approval status are complete.

Promotion requires a locked model version, reproducible feature version, reproducible split version, held-out test metrics, calibration diagnostics, top-k diagnostics, and documented rollback.
