# Limitations And Prohibited Uses

The data is CMS DE-SynPUF synthetic Medicare claims data and should be treated as a project dataset, not a live production population.

The model predicts next-year top-decile cost status from prior-year beneficiary-level features. It does not establish treatment effects or causal savings.

The model must not be used as the sole basis for coverage, benefits, pricing, reserving, clinical decisions, or adverse consumer decisions.
